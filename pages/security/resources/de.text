h2. Anleitungen zum Thema Sicherheit

Empfohlen:

* [[EFF-Anleitung für Überwachungs-Selbstverteidigung -> https://ssd.eff.org/]] (English).
* [[Anleitung für digitale Selbstverteidigung -> https://guide.boum.org/]] (French, bitte hilf' beim Übersetzen).
* [[Sicherheit 'in-a-box' -> https://security.ngoinabox.org/]] enthält viele praktische Anleitungen von [[Front Line -> https://www.frontlinedefenders.org/]] und [[Tactical Technology Collective -> http://www.tacticaltech.org/]]
* FOSS Anleitung zum [[Umgehen von Internetzensur -> https://www.flossmanuals.net/bypassing-censorship/]] ist eine umfassende 240-seitige Anleitung über Umgehungstechnologie.
* FOSS Anleitung über [[Gründlagen der Internetsicherheit -> https://www.flossmanuals.net/basic-internet-security/]]
* [[Internetüberwachungs-Wiki -> http://en.cship.org/wiki]] enthält viele Informationen über das Umgehen von Internetzensur (English).
* [[Riseup Zine: Digitale Sicherheit für Aktivisti!->https://web.archive.org/web/20160306044630/https://zine.riseup.net/]]

Veraltet, aber manchmal hilfreich:

* [[security.resist.ca -> http://security.resist.ca]] hilft Aktivisti, in einer oppressiven Welt sicher zu bleiben.
* [[APC Sicherheitsdokumente -> http://secdocs.net/manual/lp-sec/]] ist eine Serie von Anleitungen über Informationssicherheit und Online-Sicherheit für Bürgerrechtsorganisationen.
* [[Anleitung für Email-Sicherheit -> http://www.activist.ca/guide/encrypt.html]] zum Nutzen von Verschlüsselung und digitale Signaturen
* [[Computer-Sicherheit für durchschnittliche Aktivisti -> http://lists.madimc.org/~infoshop/PDF/compsecurity.pdf]] (PDF).
* [[Eine Anleitung über Aktivismus im Internet -> http://www.backspace.com/action/]]
